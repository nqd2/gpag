# 10 Steps to Start on BRAIN

**10 Steps to Start on BRAIN**

- Watch the [“Introduction to Alphas”](https://platform.worldquantbrain.com/learn/courses/introduction-alphas) training series
 - Read the [starter pack series](https://platform.worldquantbrain.com/learn/documentation#:~:text=Documentation-,Discover%20BRAIN,-4%20Articles) under “Discover BRAIN” section
 - Simulate three simple formulas using the data field: close and the operator: rank
 - Understand the simulation settings, starting with [delay and neutralization](https://platform.worldquantbrain.com/learn/documentation/create-alphas/simulation-settings) .
 - Click the Example button on the bottom left of the [Simulate](https://platform.worldquantbrain.com/simulate) page for examples. Try to improve these expressions using the Hint and then submit
 - Understand the [Simulation Results](https://platform.worldquantbrain.com/learn/documentation/interpret-results/parameters-simulation-results) and submission tests.
 - Simulate Alphas discussed in the "Create Alphas" sections [Run your First Alpha](https://platform.worldquantbrain.com/learn/documentation/create-alphas/running-your-first-alpha) and [Another Sample Alpha](https://platform.worldquantbrain.com/learn/documentation/create-alphas/another-sample-alpha) . Pay attention to the settings.
 - Check the [Events](https://platform.worldquantbrain.com/events) page to attend training webinars.
 - Read the [BRAIN TIPS series](https://support.worldquantbrain.com/hc/en-us/search?filter_by=community&query=%5Bbrain+tips&utf8=%E2%9C%93) and the [Beginner](https://support.worldquantbrain.com/hc/en-us/search?content_tags=01GVTJQR8518KQMHP1JMCZ4Y99&utf8=%E2%9C%93) posts in FAQs/ Forums
 - Try to create Alphas using available [Price volume](https://platform.worldquantbrain.com/data/data-sets/pv1) data fields or and, [Operators](https://platform.worldquantbrain.com/learn/data-and-operators/operators)
