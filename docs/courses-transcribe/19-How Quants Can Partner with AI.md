# 19-How Quants Can Partner with AI

Programs like AlphaGo and AlphaZero made history when they defeated grandmasters in the respective games of Go and Chess. This victory over the world's top players, which many thought would take decades to achieve, underlines the potential of machines to surpass humans in complex tasks.

Now let's see how Quantz can partner with advanced statistical models, machine learning and deep learning models to help enhance data on brain. The models can predict missing historical data, enhancing the original data's coverage.

The models can help make intermediate predictions on low frequency data sets, such as enhancing quarterly sales metrics in brain's fundamental data set to weekly or daily frequency. This can boost alpha turnover and performance. Quantz can build new alphas by predicting returns one or five days ahead, or improve alphas that use past information. We can combine multiple data fields using advanced models into fewer predictive features that contain the essence of all the relevant data.

This approach can help harness the massive data growth we have seen. It also addresses the high-dimensionality problem of many data sets, where data fields outnumber the observations, a situation that can result in overfitting.
